
# Images and Button Events Android App

## Overview

This Android application was created as part of the ECPI University Mobile App Development course.

The project demonstrates:

- Image resource management
- Android XML layouts
- Button click event handling
- Activity navigation using Intents
- Android Studio project structure
- Emulator testing and deployment

## Technologies Used

- Kotlin
- Android Studio
- Android SDK 37
- ConstraintLayout

## Features

- Main application screen
- Recipe screen displaying Bruschetta ingredients and directions
- Navigation between activities using a button click event
- Image asset integration

## Skills Demonstrated

- Android UI development
- Event-driven programming
- Intent-based navigation
- Resource management
- Android SDK troubleshooting
- Git and GitHub workflow
